# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: **security@ctrahey.com**

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This project makes HTTP requests to user-configured remote services. Please report any issues related to:

- Unintended data exposure in HTTP payloads
- Path traversal or injection in configuration handling
- Cache poisoning or cache path manipulation
- Any other security concern
